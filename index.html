Что такое PSMC?

Честно о сервере

PSMC — это простой, чистый сервер там нет модов и плагинов которые влеяют на гемплей. Здесь нет приватов, нет экономики, нет донатов. Только ванильный Minecraft в его чистом виде.

Чистый ванильный сервер
Без приватов и защиты
С голосовым чатом
Бесплатно и навсегда

Как подключиться

1. Запустите Minecraft Java

Версия 1.21.11

2. Добавьте сервер

Меню → Сетевая игра → Добавить сервер

3. Введите адрес

psmc_java.creepers.sbs

4. Играйте!

Присоединяйтесь к серверу

Поддержка Bedrock Edition

Сервер доступен для Bedrock (PE/Xbox/PS/Switch) через порт 19132

1. Откройте Minecraft Bedrock

Версия 1.21.132 может быть можно и выше

2. Перейдите во вкладку "Серверы"

И нажмите "Добавить сервер"
